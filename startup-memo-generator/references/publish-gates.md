# Publish Gates

## Hard Blocks

- Dead links in rendered memo.
- Missing source snapshot.
- Missing or invalid access mode.
- Broken auth route/template.
- Broken analytics route/template.
- Broken mobile or desktop render.
- Runtime route errors.

## Warnings

- PII-sensitive content.
- PR-sensitive content.
- Missing stakeholder locale.
- Broad `pin-only-with-email-capture` distribution.
- A/B reports with small sample sizes.

Warnings must be visible in the gate report and audit log. Warnings do not block publish unless the user explicitly upgrades a warning to a hard gate.
